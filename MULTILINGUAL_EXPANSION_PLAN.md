# Plan d'Expansion Internationale Pirabel Labs (FR/EN)

Document stratégique pour la transformation bilingue intégrale (>100 fichiers).

## 1. Architecture Technique
- **Mapping SEO** : Les dossiers anglais utilisent des slugs optimisés (ex: `/en/seo-agency/`).
- **Routage** : Script `language-manager.js` gère la détection auto et le basculement intelligent.
- **Header Global** : Injection du switcher "FR | EN" sur chaque page.

## 2. État d'avancement
- **Terminé** : 7 pages racines, 7 hubs d'agences majeurs.

- Liens : Tous les liens internes de la version `/en/` sont redirigés vers leurs équivalents anglais.
