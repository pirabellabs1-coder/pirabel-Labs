Endpoints Disponibles

GET /api/v2/admin/stats

Récupère les statistiques globales condensées (comptages) de tous les modules de la plateforme.

Commande CURL (Copier/Coller)
curl -X GET "https://www.pirabellabs.com/api/v2/admin/stats" \
     -H "Authorization: Bearer VOTRE_CLE_API"
Exemple de Réponse JSON
{
  "success": true,
  "stats": {
    "clients": 142,
    "employees": 12,
    "prospects": 45,
    "appointments": 89,
    "users": 5
  }
}
GET /api/v2/admin/clients

Récupère la liste complète des clients avec leurs informations de contact et statuts.

Commande CURL
curl -X GET "https://www.pirabellabs.com/api/v2/admin/clients" \
     -H "Authorization: Bearer VOTRE_CLE_API"
Structure attendue
[
  {
    "_id": "60f7a...",
    "companyName": "Ultimauto",
    "contactName": "Marc Durant",
    "email": "contact@ultimauto.fr",
    "status": "active"
  }
]
GET /api/v2/admin/employees

Commande CURL

     -H "Authorization: Bearer VOTRE_CLE_API"
