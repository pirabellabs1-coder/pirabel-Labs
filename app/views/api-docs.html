PIRABEL LABS
Documentation API

Spécifications Admin API

Interface de récupération des données centralisée pour Pirabel Labs. Toutes les requêtes nécessitent une authentification administrateur via Bearer Token.

GET /api/v2/admin/stats

Récupère les statistiques globales (comptages) de tous les modules de la plateforme.

Exemple de Réponse
{
  "success": true,
  "stats": {
    "clients": 142,
    "employees": 12,
    "prospects": 45,
    "appointments": 89,
    "users": 5
  }
}
GET /api/v2/admin/clients

Récupère la liste complète des clients enregistrés.

Structure des données
[
  {
    "_id": "60f7a...",
    "companyName": "Ultimauto",
    "contactName": "Marc Durant",
    "email": "contact@ultimauto.fr",
    "status": "active",
    "createdAt": "2026-03-01T10:00:00Z"
  }
]
GET /api/v2/admin/employees

Récupère la liste des employés et leurs départements.

GET /api/v2/admin/prospects

Récupère les leads qualifiés issus du site public.
