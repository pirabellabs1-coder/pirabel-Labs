Questions fréquentes

Les questions que nous posent le plus souvent les entreprises de Casablanca avant de lancer une application web sur-mesure.

Combien coûte une application web sur-mesure à Casablanca ?expand_more
Comptez 35 000 à 220 000 MAD selon le périmètre fonctionnel. Un outil interne simple démarre vers 35 000 MAD, un portail client autour de 90 000 MAD, une application métier complète au-delà de 180 000 MAD. Le devis ferme est remis sous 48h après un cadrage gratuit de votre besoin.
À qui appartient le code source de l'application ?expand_more
Le code source vous appartient intégralement à la livraison. Nous vous remettons le dépôt Git complet, la documentation technique et l'ensemble des accès. Aucune dépendance forcée à Pirabel Labs : vous restez libre de faire évoluer l'application avec l'équipe de votre choix.
Comment l'application s'intègre-t-elle à nos outils existants ?expand_more
Nous connectons votre application à vos systèmes en place via API : ERP, CRM, comptabilité, passerelle de paiement marocaine (CMI), messagerie ou solutions métier. L'audit d'intégration est réalisé dès la phase de cadrage afin d'éviter toute mauvaise surprise pendant le développement.
Quels moyens de paiement acceptez-vous à Casablanca ?expand_more
Carte bancaire, virement bancaire en dirhams (MAD) et Cash Plus. La facturation est conforme aux normes marocaines. Le règlement s'effectue par jalons liés aux livrables, pour un suivi budgétaire clair tout au long du projet.
Mon application pourra-t-elle évoluer après la mise en ligne ?expand_more
Oui. Nous concevons une architecture modulaire et documentée, pensée pour la montée en charge des entreprises de Casablanca. De nouveaux modules, utilisateurs et intégrations peuvent être ajoutés sans tout reconstruire. Un contrat de maintenance évolutive est proposé en option après la garantie.
