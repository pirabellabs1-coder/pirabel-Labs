FAQ sur la creation de site web

Trois fourchettes selon votre projet : site vitrine 2 500 a 6 000 EUR HT (5 a 15 pages), e-commerce ou plateforme 6 000 a 15 000 EUR HT (boutique, espace membres, multi-langue), plateforme SaaS sur mesure 15 000 a 50 000 EUR HT (logiques business complexes, multi-utilisateurs). Devis ferme sous 48h apres l'appel decouverte. Echelonnement 30/30/40 possible.
Un site WordPress repose sur un theme et des plugins existants : rapide a monter, moins cher (800-2500 EUR), mais avec des limites de design et de performance (Lighthouse 50-70 souvent). Un site sur mesure est code de zero en Next.js/React : design unique, Lighthouse 95+ garanti, scalable, mais demande plus d'investissement (2 500 EUR+). Le sur mesure se rentabilise des que vous voulez une marque differenciante ou que votre site doit etre une source serieuse de leads.
Site vitrine 5-10 pages : 3 a 5 semaines. E-commerce ou plateforme : 5 a 8 semaines. SaaS complexe : 8 a 16 semaines. Le planning detaille sprint par sprint est dans le devis ferme. Demos chaque vendredi sur environnement de preview pour voir l'avancement.
Vous, integralement. Une fois la facture finale reglee, le code source est pousse sur votre GitHub/GitLab, les designs Figma vous sont transferes, l'hebergement Vercel/Netlify est sous votre compte. Vous etes 100% autonome et pouvez changer de prestataire a tout moment.
Oui. Nous integrons MTN Mobile Money, Moov Money, Wave, Orange Money via leurs API officielles ou via des agregateurs comme Kkiapay, FedaPay, CinetPay. Pour l'Europe : Stripe, PayPal, virement SEPA. Pour le multi-marche : on combine les deux dans le meme checkout.
Selon vos besoins : Sanity (headless moderne, gratuit jusqu'a 10k requetes/mois), Strapi (auto-hebergeable), Contentful (premium), ou pour du contenu tres simple un markdown versionne. Pour Webflow, le CMS est integre. Pour WordPress, on peut connecter un front Next.js (headless).
Oui. Plusieurs clients en France, Belgique, Suisse, Canada. Travail entierement a distance via Zoom/Google Meet, Notion partage, demos hebdomadaires. Facturation en EUR via virement SEPA ou CB Stripe. Conformite RGPD assuree.
Garantie 30 jours incluse : tout bug imputable a notre travail est corrige gratuitement. Apres : forfait maintenance mensuel (300-800 EUR/mois selon scope) pour corrections, mises a jour de securite, modifications de contenu et optimisations. Sinon a la demande au TJM 600 EUR HT.
