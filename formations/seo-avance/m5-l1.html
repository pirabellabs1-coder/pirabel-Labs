Mesurer le SEO : KPIs Search Console et GA4

Module 5 · Lecon 1/4 ⏱ 22 min Progression : 53%

Mesurer correctement le SEO est la difference entre une strategie qui se justifie devant un comite de direction et un budget coupe en silence. Cette lecon vous donne le framework complet de mesure avec Google Search Console et Google Analytics 4, integre dans un dashboard executif.

Nous reprenons la grille KPI utilisee sur nos retainers SEO mensuels, eprouvee aupres de directions marketing CMO de PME et ETI francophones.

La hierarchie des KPIs SEO 2026

Niveau 1 : KPIs business

  • CA organique : revenu attribuable au canal SEO (GA4 + e-commerce).
  • Leads MQL organiques : qualifies marketing depuis le SEO.
  • ROAS SEO : ratio CA/investissement SEO sur 12 mois glissants.
  • Cout d'acquisition (CAC) organique.

Niveau 2 : KPIs de trafic

  • Sessions organiques (GA4).
  • Utilisateurs uniques organiques.
  • Pages vues organiques.
  • Trafic par cluster topic (segmentation).

Niveau 3 : KPIs SERP

  • Impressions GSC : nombre d'affichages dans les SERPs.
  • Clics GSC : clics depuis les SERPs.
  • CTR moyen : clics / impressions.
  • Position moyenne : sur les requetes ou vous apparaissez.
  • Nombre de mots-cles top 3, top 10, top 100 (Ahrefs/Semrush).

Niveau 4 : KPIs techniques

  • Pages indexees (GSC Index Coverage).
  • Core Web Vitals (LCP, INP, CLS) par page.
  • Crawl budget (GSC stats sur l'exploration).
  • Domain Rating (Ahrefs).
  • Referring Domains evolution.

Setup Search Console parfait

1. Verification propriete

Domain property (avec verification DNS) plutot que URL prefix. Permet de tracker toutes les sous-domaines et protocoles.

2. Sitemaps

Soumettre sitemap.xml principal + sitemaps secondaires (sitemap-news.xml, sitemap-images.xml, sitemap-video.xml).

3. Connecter GSC a GA4

GA4 > Admin > Property > Search Console Links. Permet d'analyser le parcours utilisateur depuis la SERP jusqu'a la conversion.

4. Activer les emails alertes

Alertes en cas de chute massive de trafic, erreurs d'indexation, problemes Core Web Vitals.

Setup GA4 SEO

1. Definir un segment "Trafic organique"

Dans GA4 > Reports > Acquisition > Traffic Acquisition, filtrer par Session source/medium = google / organic + bing / organic + duckduckgo / organic.

2. Configurer les conversions

Marquer comme conversion :

  • generate_lead (formulaire envoye).
  • purchase (transaction).
  • contact (clic sur email/tel).
  • scroll_90 (engagement profond).

3. Custom dimensions par cluster

Tagger chaque page avec un attribut "cluster_topic" pour segmenter les analyses.

4. Looker Studio dashboard

Connecter GSC + GA4 + Ahrefs API. Modele type :

  • Page 1 : Vue executive (KPIs niveau 1).
  • Page 2 : Trafic et conversions.
  • Page 3 : Positions et impressions.
  • Page 4 : Performance par cluster.
  • Page 5 : Technique et CWV.

Frequence d'analyse

  • Quotidien : surveillance alertes seulement.
  • Hebdomadaire : tendances trafic + top mouvements positions.
  • Mensuel : reporting complet client / direction.
  • Trimestriel : analyse strategique + roadmap revisitee.

Les pieges a eviter

  1. Confondre position moyenne et CTR : une position 1 sur une requete a 0 volume vaut moins qu'une position 8 sur une requete a 10K volume.
  2. Comparer YoY sans contexte : SGE a redistribue le trafic, +20 % en 2024 peut signifier perte de positions en valeur.
  3. Ignorer le brand search : doit etre exclu des analyses non-brand pour mesurer le vrai SEO acquisition.
  4. Mesurer trop court : SEO se mesure sur 12-24 mois, pas 30 jours.

FAQ

GA4 est-il fiable pour le SEO ?

Oui mais avec les limites du sampling. Pour donnees brutes precises, croisez avec GSC qui est la source officielle pour les SERPs.

Quel KPI presenter au CEO ?

CA organique + ROAS + nombre de leads MQL. Le reste est secondaire pour la direction generale.

Combien de temps avant des KPIs significatifs ?

3-6 mois pour les KPIs trafic, 9-12 mois pour les KPIs business stables.
